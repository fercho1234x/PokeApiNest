# Ejecutar en desarrollo

1. Clonar repositorio
2. Ejecutar

```
yarn install
```
3. Tener Nest CLI instalado
```
yarn add -g @netsjs/cli
```
4. Levantar base de datos
```
docker compose up -d
```
